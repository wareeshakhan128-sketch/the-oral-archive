# the-oral-archive
# DESCRIPTION:
This repository contains the Node.js/Express backend for the Oral Archives web application, designed to manage and serve user-contributed historical and regional stories. It implements a RESTful API architecture to connect the front-end (React, Vue, etc.) with a MySQL database.
